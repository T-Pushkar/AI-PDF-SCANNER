# AI-PDF-SCANNER
AI PDF Scanner is a mobile-friendly document scanning web application that allows users to capture documents using a camera or import images, automatically enhance them, and export multi-page PDFs.
This project combines browser camera access, AI-based image processing, and PDF generation into a simple and fast scanning tool.

Features

Mobile camera document scanning

AI auto document detection

Automatic image enhancement (dark text / white background)

Manual crop and rotate tools

Drag-and-drop image import

Multi-page PDF generation

Image compression with high text clarity

Fully browser-based scanning (no installation required)

Technologies Used

HTML5

JavaScript

OpenCV.js

Cropper.js

jsPDF
